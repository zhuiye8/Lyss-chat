## 📝 开发规范

### 代码风格
- TypeScript 严格模式
- 函数式组件 + Hooks
- 自定义 Hook 封装复用逻辑
- CSS-in-JS (styled-components 或 emotion)

### 组件规范
- 组件名: `PascalCase`
- 文件名: `index.tsx` (组件目录)
- Props 接口: `ComponentNameProps`
- 事件处理: `handleXxx` 命名

### 状态管理
- 服务端状态: React Query
- 客户端状态: Zustand
- 表单状态: Ant Design Form
- 路由状态: UmiJS 路由
